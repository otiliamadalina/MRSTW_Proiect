## Website MADGYM

Website MADGYM is a full-featured web application for managing a fitness gym website.  

---

## Features

- **Authentication and Registration**  
  Users can create accounts and log in securely using credentials.

- **Membership Management**  
  Users can view and select from multiple membership plans with price and duration.

- **Discount Code Application**  
  Discounts can be applied at checkout using valid codes.

- **Online Payment Integration**  
  Users can complete transactions using card details.

- **Admin Management**  
  Admins can add, update, or remove entities via the dashboard.

---

## Technologies Used

### Frontend
- **HTML / CSS / JS**
- **Bootstrap 5** – modern UI framework  
- **jQuery** – DOM manipulation + AJAX for real-time updates

### Backend
- **ASP.NET MVC**  
- **C#** – business logic  
- **Entity Framework** 

### Database
- **SSMS**  
- Database schema includes the following tables:
  - `Users`
  - `Memberships`
  - `DiscountCodes`
  - `Orders`

---

## Requirements

- Visual Studio
- .NET Framework 4.8
- SQL Server
- NuGet packages for:
  - EntityFramework
  - Microsoft.AspNet.Mvc
